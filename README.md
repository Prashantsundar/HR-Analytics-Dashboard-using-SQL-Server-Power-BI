# 📊 Employee Performance Analysis (SQL + Power BI)

## 🚀 Project Overview

This project is an end-to-end Employee Performance Analysis using SQL Server and Power BI.
It focuses on cleaning raw HR data, analyzing employee performance, and building an interactive dashboard to generate actionable insights.

---

## 🎯 Objectives

* Clean and preprocess employee dataset using SQL
* Analyze employee performance and ratings
* Identify top and underperforming employees
* Evaluate department-wise performance
* Build an interactive Power BI dashboard

---

## 🛠️ Tools & Technologies

* SQL Server Management Studio (SSMS)
* SQL (T-SQL)
* Power BI
* Excel / CSV Dataset

---

## 📂 Dataset

The dataset includes:

* Employee details
* Department and job roles
* Performance scores
* Employee ratings

---

## 🧹 Data Cleaning (SQL)

* Removed NULL values
* Created a cleaned dataset
* Standardized key columns for analysis

---

## 📊 Analysis Performed

* Top performing employees
* Performance distribution
* Department-wise average ratings
* High-performing departments
* Underperforming employees

---

## 📈 Power BI Dashboard Features

* Interactive filters (Department, Job Role)
* KPI cards (Average Rating, Total Employees)
* Performance distribution charts
* Department-wise comparison
* Top performers visualization

---

## 🔍 Key Insights

* Identified departments with highest performance ratings
* Highlighted top-performing employees
* Detected underperforming areas for improvement
